## 使用方式

建議使用本模式進行 ASCI Art Installer 的安裝

請輸入以下指令到您的終端機

```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/m85091081/asciiartcollection/shell-script/scripts/loader.sh )"
```
